THE CURRICULUM

What You'll Learn

We've structured this workshop to focus strictly on modern standards and core principles, cutting through the noise to get you coding fast.

1. HTML5 Structure

Learn the skeletal structure of the modern web. Master semantic markup, SEO best practices, page hierarchy, and clean structure to make your web elements highly accessible and organized.

2. CSS3 & Responsiveness

Style your web pages with modern layout systems like CSS Grid and Flexbox. Design responsive components that scale flawlessly from mobile screens to massive high-res desktop displays.

3. JavaScript Magic

Inject life and interactivity into your website. Understand the Document Object Model (DOM), handle user events, trigger custom animations, and write dynamic scripting for modern UI/UX workflows.

From Scratch to Live Server

The workshop ends with a live build. We won't just keep the code on your local computer; we will show you how to host and deploy your newly designed site onto a professional, public cloud service (Netlify / Vercel / GitHub Pages) so you can share it instantly with the world.

ROADMAP

3-Day Live Schedule

A detailed daily breakdown of topics, exercises, and projects. Click on the tabs below to explore each day's agenda.

10:00 AM - 11:30 AM

Introduction & Setup

Welcome address and introduction to Anti-Gravity concepts. Setting up our development environment (Visual Studio Code, extensions, and workspace files).

11:45 AM - 01:15 PM

Understanding HTML Tags & Semantics

An introduction to page layouts: headers, sections, articles, tables, lists, and semantic tags. Structuring the prototype of our workshop landing page.

02:00 PM - 04:00 PM

Hands-On Lab: HTML Page Structure

Building our very first basic html structure from scratch. Students will create forms, inputs, navigation, and embed media files correctly.

10:00 AM - 11:30 AM

CSS Core Concepts & Selectors

Introduction to the CSS Box Model, specificity, inline vs. external styles, custom fonts, and applying colors, borders, margins, and padding variables.

11:45 AM - 01:15 PM

Modern Layouts: Flexbox and CSS Grid

Designing modern page layouts. Stacking elements vertically on mobile and spreading them horizontally on desktop using Flexbox properties and Grid tracks.

02:00 PM - 04:00 PM

Media Queries & Mobile-First Styling

Applying custom media breakpoints. Designing smooth typography scaling, hiding/showing navigation bars, and making forms fluidly resize for smartphones.

10:00 AM - 11:30 AM

Introduction to JS & DOM Manipulation

Understanding variables, arrays, simple event listeners, click handlers, and dynamically updating texts or classes within the HTML page via vanilla JavaScript.

11:45 AM - 01:15 PM

Making Interactive Components

Coding components like an interactive mobile menu drawer, sliding FAQ accordions, and custom modal windows for registration validation.

02:00 PM - 04:00 PM

Live Deployment Workshop

Taking our fully styled, interactive landing page, committing it, and deploying it live on a web server in real-time. Project wrap-up and certification guidelines.

HELP CENTER

Frequently Asked Questions

Have queries? We've compiled quick answers to the most common questions asked about this workshop.

Who is this workshop for?
This workshop is tailored specifically for beginners with absolute zero programming background, as well as students looking to brush up on their fundamental concepts in modern HTML, CSS layouts, and simple JavaScript interactivity.
Are there any prerequisites or software required?
All you need is a laptop (Windows, Mac, or Linux) with an active internet connection and a modern browser (Google Chrome or Microsoft Edge). We will help you set up Visual Studio Code (a free text editor) on Day 1.
Is the workshop certified?
Yes! All participants who successfully complete the 3-day roadmap and submit their deployed live project links will receive a verified certificate of completion issued by Anti-Gravity.
How much does the workshop cost?
This workshop is entirely sponsored by Anti-Gravity. Registration is 100% free, but seats are strictly limited to ensure personal mentoring for all attendees.
